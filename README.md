# roc-pg

Interface with PostgreSQL databases from pure Roc.

## Features

- [x] Simple queries
- [x] Decoding results
- [x] Parameter bindings
- [ ] Authentication
  - [ ] Cleartext password
  - [ ] MD5 password
  - [ ] SASL
  - [ ] GSS
  - [ ] SSPI
- [ ] Cursors
- [ ] Prepared statements
- [ ] SSL
- [ ] Connection pooling
- [ ] Bulk copying
- [ ] Notifications (listen/notify)
- [ ] Notices
